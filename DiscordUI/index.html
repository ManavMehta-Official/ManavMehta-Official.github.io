Introduction

Documentation and examples for how to use DiscordUI's included Discord themed user interface. Note: DiscordUI works only on chromium engine based browsers.

# Quick Start

  • manifest.json

  • background.js

  • inject.css

  • styles.css

  • ext_gen.css

  • ext_1.css

  • ext_2.css

  • login.css

  • misc.css

  • contact.css

  • popup.html
